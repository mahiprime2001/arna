# console — Tauri app (admin)

The desktop app the admin/owner runs to connect to stores.

- Device list (your stores) + **Connect**.
- Control window: live screen, mouse/keyboard capture, multi-monitor.
- Right-side **file panel**: drag-drop to/from real local folders.
- Chat; later **Meet** (audio/video + screen share).
- Opens directly, or via deep link: `arnaremote://connect?store=<id>&ticket=<signed>`.

**Depends on:** `../core`. **Status:** not yet implemented (begins Phase 1).
